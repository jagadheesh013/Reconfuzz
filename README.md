# Reconfuzz
Command line tool for subdomain enumeration and directory fuzzing
This tool is developed using python.

## Installation:
1. `git clone https://github.com/jagadheesh013/Reconfuzz.git`
2. `cd Reconfuzz`
3. `pip install -r requirements.txt`

## How to run:
`python Rconfuzz.py -m <sub/dir> -u <example.com> -w <path/to/wordlist>`
